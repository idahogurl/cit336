Clone a GitLab Repository

Prerequisites:

  • MAMP for Windows or MAMP for Mac
  • GitLab Account
  • Visual Studio Code

Copy GitLab repository URL

  1. Log into your GitLab account. If you are already logged go to the upper-left corner and click the GitLab logo. This will take you to your project list.
  2. When the project list loads click on your "acme" project.
  3. To the right, click the blue "Clone" button
  4. A small popup will display. Click the copy button next to the 2nd textbox.
  5. You now have the URL you need to connect Visual Studio Code to GitLab.

Connect VS Code to GitLab

In the next video when GitHub is mentioned, replace it with GitLab and try to follow the steps. The steps may not perfectly align, but they are very similar. These are the differences.

  • Instead of pasting a GitHub URL, use the GitLab URL that you just copied.
  • Your local folder should be the one you created when we setup MAMP.
  • The GitLab repository we are cloning is empty, so there will not be any files in your folder.
